# Prototype 0.1.1 Notes

Last updated: 2026-05-01

## Purpose

Prototype 0.1.1 is the first readability checkpoint after the verified Prototype 0.1 Combat Sandbox.

It does not add new gameplay systems. It makes the existing combat sandbox easier to understand while staying primitive.

## Verified In Play Mode

- Movement still works.
- Dash still works.
- Basic Attack, Cleave, Stomp, and Rage still work.
- Enemies can still be hit and killed.
- Archer can be hit and no longer escapes the arena.
- Brute still drops Echo Axe.
- Echo Axe can still be picked up.
- Echo Axe still modifies Cleave into a shockwave.

## Readability Improvements

- Darker dungeon mood.
- Stronger floor/wall contrast.
- Clearer player and enemy silhouette colors.
- Simple torch-like placeholder lights.
- Light fog and darker camera backdrop.
- Enemy name labels.
- Simple enemy health bars.
- Smaller top-left prototype controls overlay.
- Compact skill cooldown text near mobile buttons.
- Clearer Echo Axe pickup label.
- Basic Attack slash streak.
- Basic Attack hit spark.

## Accepted Placeholder Quality

These are still placeholder:

- Primitive geometry.
- Debug skill markers.
- Primitive hit sparks.
- Primitive enemy labels and health bars.
- Placeholder mobile HUD.
- Placeholder lighting/fog.
- No animations.
- Audio was not part of Prototype 0.1.1; later passes added placeholder generated audio.
- No final VFX.

## Known Feel Gaps

- Enemy death still needs clearer feedback.
- Loot drop should pulse or flash more strongly when it appears.
- Dash still needs real VFX/final audio later.
- Enemy telegraphs are still primitive.
- UI is readable enough for testing but not final mobile UI.
- Combat numbers are not final balance.

## Next Smallest Step

Prototype 0.1.2 should focus on enemy death and loot-drop clarity only:

- Add a simple death puff or flash.
- Add a short fade/shrink before enemy removal if simple.
- Make loot drop pulse or glow when it appears.
- Do not add new loot systems.
- Do not add inventory.
- Do not import art assets.
- Do not add new enemies or skills.
