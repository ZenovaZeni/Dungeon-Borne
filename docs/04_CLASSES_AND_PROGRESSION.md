# Classes and Progression

## Base Classes for First Full Demo

### Fighter
Durable melee class with weapons, shields, cleaves, stuns, and rage/guard paths.

Evolutions:
- Guardian
- Berserker

### Rogue
Fast class with poison, crits, traps, dash resets, and burst windows.

Evolutions:
- Assassin
- Trickster

### Mage
Ranged caster with elemental spells, curses, summons, and area control.

Evolutions:
- Elementalist
- Warlock

## Class Evolution Rules

At a certain level or after finding a class relic, the player chooses an evolution path.

Evolution should:
- change skill visuals
- unlock new skills
- modify existing skills
- shift stat growth
- create a new playstyle

## Example: Fighter

Starting skills:
- Slash
- Dash
- Cleave
- Stomp
- Battle Cry

Guardian path:
- shield scaling
- defensive counters
- taunt effects
- holy/earth shockwaves
- survivability

Berserker path:
- rage resource
- bleed
- lifesteal
- risky damage boosts
- faster attack chains

## Progression Philosophy

Levels should give:
- stat increases
- skill points
- class unlocks
- passive choices

Gear should give:
- stat changes
- build direction
- skill mutations

Class evolution should give:
- identity
- new strategy
- replayability
