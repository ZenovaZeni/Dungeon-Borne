# Loot System

## Loot Philosophy

Loot should make the player say:

"Wait, I want to try a build around this."

## Rarity Tiers

### Common
Basic stat increases.

### Rare
Stats plus one simple bonus.

### Epic
Stats plus a build effect.

### Legendary
Changes a skill or creates a new playstyle.

## Item Slots

V1:
- weapon
- offhand/shield
- helmet
- chest/armor
- ring
- amulet

Later:
- boots
- gloves
- relic
- companion gear

## Skill-Mutating Item Examples

### Echo Axe
Cleave sends a shockwave forward.

### Mirrorstep Boots
Dash leaves a shadow clone that attacks once.

### Ember Crown
Fire skills burn longer and spread on kill.

### Gravebell Staff
Summons explode on death and apply curse.

### Saintwall Shield
Perfect blocks release a healing pulse.

### Bloodfang Blade
Bleed ticks faster below 40% enemy health.

## Visual Gear Rules

Do not show every piece of gear in v1.

Show:
- weapon
- shield/offhand
- helmet maybe
- armor color/theme
- aura/VFX for class or legendary effects

This gives visual progression without exploding art scope.
