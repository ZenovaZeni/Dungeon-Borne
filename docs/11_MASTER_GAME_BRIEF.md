# Master Game Brief

## North Star

Dungeon Borne is a mobile-first low-poly dark fantasy action RPG about descending through dangerous dungeon chapters, finding loot that changes how skills behave, and evolving a hero class into stronger identities.

The core loop is:

```text
Fight -> Loot -> Upgrade -> Evolve class -> Descend deeper
```

## Target Feel

The game should blend:

- Diablo-style dungeon runs and loot excitement.
- Final Fantasy Tactics-style class identity and evolution choices.
- WoW-inspired cooldown timing, counterplay, and readable enemy attacks.
- Mobile-friendly action combat built for thumbs first, with PC/controller support from the same input architecture.

Combat must feel good in one room before the project expands.

## Player Role

The player controls one hero.

Prototype 0.1 focuses only on the Fighter:

- Basic slash
- Dash
- Cleave
- Stomp
- Rage/ultimate placeholder

Future classes and evolutions are important to the full game, but they should not be built until the first combat sandbox is testable and fun.

## What Matters Most

1. Responsive movement.
2. Readable combat.
3. Skills that feel different from each other.
4. Loot that changes gameplay, not only stats.
5. Strong class identity.
6. Mobile readability.
7. Expandable dungeon chapters.

## Commercial Direction

The first commercial structure should be chapter-based dungeon expansions.

Good expansion content:

- New dungeon chapter
- New enemy faction
- New boss
- New gear and legendary modifiers
- New class relics or evolutions
- New story continuation

Avoid pay-to-win. Paid content should not break future arena fairness.

## Future Systems, Not Now

Allowed later:

- Companions
- Arena/PvP
- More classes
- Class evolution trees
- Full inventory and equipment
- Story scenes
- Dungeon chapters

Not part of Prototype 0.1:

- Companions
- PvP
- Multiplayer
- Open world
- Monetization
- Full inventory
- Full story
- Large asset-pack integration

## Prototype Rule

Do not build the full game yet.

Prototype 0.1 is successful only when one room proves:

- Movement works.
- Dash works.
- Basic attack works and is readable.
- Cleave works and is readable.
- Stomp works and is readable.
- Rage placeholder works and is readable.
- Three enemy types behave differently.
- Enemies take damage and die.
- Damage numbers are readable.
- Loot drops and can be picked up.
- Echo Axe changes Cleave into a shockwave.
