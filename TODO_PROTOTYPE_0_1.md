# Prototype 0.1 TODO / Placeholder Notes

This repository is currently Unity-adoptable source structure, not a confirmed Unity project until Unity Hub creates or opens it.

## Working Target Loop

- Player movement through Unity Input System actions.
- Player basic attack, dash, Cleave, Stomp, and Rage placeholder.
- Skeleton Grunt, Archer, and Brute definitions.
- Enemy chase, melee attack, and ranged projectile behavior.
- Health, damage, death, floating damage numbers.
- Simple loot drop.
- Echo Axe legendary modifier: Cleave launches a shockwave.

## Known Placeholders

- Character, enemy, room, projectile, loot, and VFX visuals are primitive-generated placeholders.
- Rage/ultimate is intentionally only a damage placeholder for Prototype 0.1.
- Mobile UI uses Input System actions and generated on-screen button/joystick placeholders; polish is deferred.
- Enemy AI is simple chase/range behavior with no advanced telegraphs yet.
- Loot has no full inventory. Picking up Echo Axe only enables its modifier on the player.
- Audio hooks are not implemented yet; docs say placeholder audio matters, so this should be the next sensory pass after the loop compiles.

## Do Not Add Yet

- Companions
- PvP or arena ranking
- Full inventory/equipment
- Story systems
- Open world
- Monetization
- Extra classes beyond Fighter
